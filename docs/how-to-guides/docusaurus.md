# Docusaurus

## publish

Automatically on merge to `trunk` branch via GitHub actions

## development

```bash
npm run start
```

## GitHub actions

[helpful setup link](https://evantay.com/blog/docusaurus-gh-action/)


## Custom domain

Create CNAME file in the `/static` directory which contains the custom domain path

```
docs.lockdev.com
```
